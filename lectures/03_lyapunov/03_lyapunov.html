Lyapunov's Direct Method

The basic philosophy of Lyapunov's direct method is the mathematical extension of a
fundamental physical observation: if the total energy of a mechanical (or electrical)
system is continuously dissipated, then the system, whether linear or nonlinear, must
eventually settle down to an equilibrium point. Thus, we may conclude the stability of
a system by examining the variation of a single scalar function V(x)V(\mathbf{x})V(x).

Specifically, let us consider the nonlinear mass-damper-spring system:

my¨+b∣y˙∣y˙+k0y+k1y3=0m\ddot{y}+b|\dot{y}|\dot{y} +k_0 y + k_1 y^3 = 0my¨​+b∣y˙​∣y˙​+k0​y+k1​y3=0

Assume that the mass is pulled away from the
natural length of the spring by a large distance, and then released.
Will the resulting motion be stable? A physical intuition said that it should be.

However, lets take a bit different approach. Consider the following function (energy):
V(y,y˙)=12my˙2+12k0y2+14k1y4V(y, \dot{y}) = \frac{1}{2}m\dot{y}^2 + \frac{1}{2}k_0 y^2 + \frac{1}{4}k_1 y^4V(y,y˙​)=21​my˙​2+21​k0​y2+41​k1​y4

Physically, it is easy to see that the mass must finally settle down at the natural length
of the spring, because it is subjected to a non-zero spring force at any position other
than the natural length.

Comparing the definitions of stability and mechanical energy, one can easily see some
relations between the mechanical energy and the stability concepts described earlier:

  • zero energy corresponds to the equilibrium point (y=0,y˙=0y = 0, \dot{y}=0y=0,y˙​=0)
  • asymptotic stability implies the convergence of mechanical energy to zero
  • instability is related to the growth of mechanical energy

These relations indicate that the value of a scalar quantity, the mechanical energy,
indirectly reflects the magnitude of the state vector.

The direct method of Lyapunov is based on a generalization of the concepts in
the above mass-spring-damper system to more complex systems. Faced with a set of
nonlinear differential equations, the basic procedure of Lyapunov's direct method is to
generate a scalar "energy-like" function for the dynamical system, and examine the time
variation of that scalar function. In this way, conclusions may be drawn on the
stability of the set of differential equations without using the difficult stability
definitions or requiring explicit knowledge of solutions

Such candidate Lyapunov function should satisfy the energy-like properties, namely be:

  • Strictly positive unless both state variables x\mathbf{x}x are zero.
  • Monotonically decreasing when the variables x\mathbf{x}x vary along system trajectories

In Lyapunov's direct method, the first property is formalized by the notion of positive definite functions (PD)

Positive Definite Functions

A scalar continuous function V(x)V(\mathbf{x})V(x) is said to be locally positive definite (LPD) in ball BR={x:∥x∥≤R}\mathcal{B}_R = \{\mathbf{x} : \| \mathbf{x}\|\leq R\}BR​={x:∥x∥≤R} if:

  • V(0)=0V(0) = 0V(0)=0
  • V(x)>0, ∀x≠0V(\mathbf{x})>0,\ \forall \mathbf{x} \neq \mathbf{0}V(x)>0, ∀x=0

If above property holds ∀x∈Rn\forall\mathbf{x} \in \mathbb{R}^n∀x∈Rn then V(x)V(\mathbf{x})V(x) is said to be globally positive definite (GPD)

Note that the above definition implies that the function VVV has a unique minimum at the origin.

For instance:

  • The V(x)=12x22+1−cos⁡x1V(\mathbf{x}) = \frac{1}{2}x_2^2 + 1-\cos{x_1}V(x)=21​x22​+1−cosx1​ is locally positive definite
  • While V(x)=12x12+12x22V(\mathbf{x}) = \frac{1}{2}x_1^2 + \frac{1}{2}x_2^2V(x)=21​x12​+21​x22​ is globally positive definite

A few related concepts can be defined similarly, in a local or global sense, i.e., a
function V(x)V(\mathbf{x})V(x) is negative definite if — V(x)V(\mathbf{x})V(x) is positive definite; V(x)V(\mathbf{x})V(x) is positive
semi-definite if V(0)=0V(\mathbf{0}) = 0V(0)=0 and V(x)≥0V(\mathbf{x}) \geq 0V(x)≥0 for x≠0\mathbf{x} \neq 0x=0 V(x)V(\mathbf{x})V(x) is negative semi-definite if - V(x)V(\mathbf{x})V(x) is positive semi-definite.

The prefix "semi" is used to reflect the possibility of VVV being equal to zero eventhough x\mathbf{x}x is not.

Global Stability via Direct Method

Assume that there exists a scalar function V(x)V(\mathbf{x})V(x) with continuous first order derivatives such that:

  • V(x)V(\mathbf{x})V(x) is positive definite
  • V˙(x)\dot{V}(\mathbf{x})V˙(x) is negative definite
  • V(x)→∞V(\mathbf{x})\rightarrow \inftyV(x)→∞ as ∥x∥→∞\|\mathbf{x}\|\rightarrow \infty∥x∥→∞

then the equilibrium at the origin is globally asymptotically stable

Example: Lyapunov Functions for LTI systems

The direct Lyapunov method can be in fact applied to linear systems as well:
x˙=Ax\dot{\mathbf{x}} = \mathbf{A}\mathbf{x}x˙=Ax

Consider the quadratic Lyapunov candidate:
V=xTPxV = \mathbf{x}^T\mathbf{P}\mathbf{x}V=xTPx

with time derivative:
V˙=xT(ATP+PA)x=−xTQx\dot{V} = \mathbf{x}^T(\mathbf{A}^T\mathbf{P} + \mathbf{P}\mathbf{A})\mathbf{x} = -\mathbf{x}^T \mathbf{Q} \mathbf{x}V˙=xT(ATP+PA)x=−xTQx

where Q\mathbf{Q}Q is P.D.

One can conclude that the LTI system is stable if there is P.D solution of matrix equation:
ATP+PA=−Q\mathbf{A}^T\mathbf{P} + \mathbf{P}\mathbf{A} = -\mathbf{Q}ATP+PA=−Q

Invariant Set Theorems and Lyapunov Functions

In practice we are always interested in the regions in which system is tend to stay forever, it may be stable equalibrium, the error funel etc. However the vannila Lyapunov theory does not directly allow us to find such regions. Here we can use the help of the powerful invariant set theorems, attributed
to La Salle as invariance principle.

The central concept in these theorems is that of invariant set

A set G\mathcal{G}G is an invariant set for a dynamical system if every system
trajectory which starts from a point in G\mathcal{G}G remains in G\mathcal{G}G for all future time

x(0)∈G⇒x(t)∈G,∀t∈R\mathbf{x}(0) \in \mathcal{G} \Rightarrow \mathbf{x}(t) \in \mathcal{G}, \forall t \in \mathbb{R}x(0)∈G⇒x(t)∈G,∀t∈R

A particular examples of invariant sets are, equilibria, limit cycles, energy level sets of conservative system and any particular solution of ODE.

In most cases, it is impractical to directly verify that a set is forward invariant or to construct forward invariant sets by examining all of the trajectories of a system. Instead, if a system has continuous solutions, we can evaluate if a set is forward invariant by checking that trajectories never leave the set through its boundary. We do this by comparing the angle between the dynamics f\mathbf{f}f and the normal to the set’s boundary. Yet again the Lyapunov functions provide a convenient tool to estimate the boundaries of invariant sets. One can easily find the the boundaries of invariance set Ω\OmegaΩ defined as ∂Ω\partial \Omega∂Ω by looking on such region {x:V(x)=c}\{\mathbf{x}:\mathbf{V}(\mathbf{x})=c\}{x:V(x)=c} where V˙(x)<0\dot{V}(\mathbf{x}) < 0V˙(x)<0

Region of Attraction

There is another very important connection between Lyapunov functions and the concept of an invariant set: any sublevel set of a Lyapunov function is also an invariant set. This gives us the ability to use sublevel sets of a Lyapunov function as approximations of the region of attraction for nonlinear systems.

For locally stable fixed point xe\mathbf{x}_exe​ the region of attraction (RoA) is the largest set R\mathcal{R}R:
x(0)∈R⇒lim⁡t→∞x(t)=xe\mathbf{x}(0) \in \mathcal{R} \Rightarrow \lim_{t\rightarrow \infty} \mathbf{x}(t) = \mathbf{x}_ex(0)∈R⇒t→∞lim​x(t)=xe​

Estimation of Region of Attraction via Sampling

The idea is to test conditions of theorem on some domain of state space, more specifically on random points xi\mathbf{x}_ixi​ within this domain. If conditions of theorem fails at least for one sample, namely V˙(xi)≥0\dot{V}(x_i) \geq 0V˙(xi​)≥0 then the level set V(x)=V(x)=ciV(\mathbf{x}) = V(\mathbf{x}) = c_iV(x)=V(x)=ci​ is not fully contained inside the region of attraction. Testing for the large amount of samples intuitevelly will yield the upper estimate of level set V(x)<c<c^uV(\mathbf{x})< c <\hat{c}_uV(x)<c<c^u​. While increasing the amount of samples xi\mathbf{x}_ixi​ one would expect for roa estimates to be tighter.

def sampling_roa(f, V, nablaV, x_bounds, N=1000):
    x_min, x_max = x_bounds
    x_range = np.array(x_max) - np.array(x_min)
    n = np.shape(x_min)[0]

    c = np.inf
    ct = []

    for i in range(N):
        x_i = x_min + x_range*np.random.rand(n)
        V_i = V(x_i)
        if V_i <= c:
            dV_i = np.array(nablaV(x_i))@ np.array(f(x_i))
            if dV_i >= 0:
                c = V_i
                ct.append(c)  
    return c, ct

Linearization based Lyapunov Candidate for RoA Estimation

For some stable fixed points, we can certify the local stability with a linear analysis, and this linear analysis gives us a candidate quadratic Lyapunov function:
V=xTPxV = \mathbf{x}^T\mathbf{P}\mathbf{x}V=xTPx

where A\mathbf{A}A is the linearization of original nonlinear dynamics nearby xe\mathbf{x}_exe​ and Q\mathbf{Q}Q is p.d matrix.

Given Q\mathbf{Q}Q one can then solve Lyapunov equation :
ATP+AP=−Q\mathbf{A}^T\mathbf{P} + \mathbf{A}\mathbf{P} = -\mathbf{Q}ATP+AP=−Q

And find region of attraction as level set {x:xTPx≤r}\{\mathbf{x}:\mathbf{x}^T\mathbf{P}\mathbf{x}\leq r\}{x:xTPx≤r}
