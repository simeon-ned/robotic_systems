Basics of System Analysis and Control Design

Concepts of Stability

Once the equlibrium or nominal motion is given, the natural question to ask is that how system will behave nearby.
This question is directly related to so called stability

Various types of stability may be discussed for the solutions of differential equations or difference equations describing dynamical systems. The one practically important type is that concerning the stability of solutions near a point of equilibrium. This may be analyzed by the theory of Aleksandr Lyapunov.

In simple terms, if the solutions that start out near an equilibrium point xe\mathbf{x}_{e}xe​ stay near xe\mathbf{x}_{e}xe​ forever, then xe\mathbf{x}_{e}xe​ is Lyapunov stable. More strongly, if x0\mathbf{x}_{0}x0​ is Lyapunov stable and all solutions that start out near xe\mathbf{x}_{e}xe​ converge to xe\mathbf{x}_exe​, then xe\mathbf{x}_{e}xe​ is asymptotically stable.

A strict definitions are as follows:

Equilibrium xe\mathbf{x}_exe​ is said to be:

  • Lyapunov stable if:
    ∀ϵ>0,∃δ>0,∥x(0)−xe∥<δ→∥x(t)−xe∥<ϵ,∀t\forall \epsilon>0,\exists\delta>0, \|\mathbf{x}(0) - \mathbf{x}_e\|<\delta \rightarrow \|\mathbf{x}(t) - \mathbf{x}_e\|<\epsilon, \quad \forall t∀ϵ>0,∃δ>0,∥x(0)−xe​∥<δ→∥x(t)−xe​∥<ϵ,∀t
  • Asymptotically stable if it is Lyapunov stable and:
    ∃δ>0,∥x(0)−xe∥<δ,→lim⁡t→∞∥x(t)−xe∥=0,∀t\exists \delta >0, \|\mathbf{x}(0) - \mathbf{x}_e\|< \delta, \rightarrow \lim_{t\to\infty} \|\mathbf{x}(t) - \mathbf{x}_e\| = 0, \quad \forall t∃δ>0,∥x(0)−xe​∥<δ,→t→∞lim​∥x(t)−xe​∥=0,∀t
  • Exponentially stable if it is asymptotically stable and:
    ∃δ,α,β>0,∥x(0)−xe∥<δ,→∥x(t)−xe∥≤α∥x(0)−xe∥−βt,∀t\exists \delta, \alpha, \beta >0, \|\mathbf{x}(0) - \mathbf{x}_e\|< \delta, \rightarrow \|\mathbf{x}(t) - \mathbf{x}_e\| \leq\alpha\|\mathbf{x}(0) - \mathbf{x}_e\|^{-{\beta}t}, \quad \forall t∃δ,α,β>0,∥x(0)−xe​∥<δ,→∥x(t)−xe​∥≤α∥x(0)−xe​∥−βt,∀t

Conceptually, the meanings of the above terms are the following:

  • Lyapunov stability of an equilibrium means that solutions starting "close enough" to the equilibrium (within a distance δ\deltaδ from it) remain "close enough" forever
  • Asymptotic stability means that solutions that start close enough not only remain close enough but also eventually converge to the equilibrium.
  • Exponential stability means that solutions not only converge, but in fact converge faster than or at least as fast as a particular known rate α∥x(0)−xe∥−βt\alpha\|\mathbf{x}(0) - \mathbf{x}_e\|^{-{\beta}t}α∥x(0)−xe​∥−βt

From Analysis to Design: Pole placement

As we have seen above, the proof of stability of a linear system is fairly straightforward and can be carried out through the analysis of eigenvalues, the natural question arises: can we transform an unstable system into a stable one by means of control?

To answer this let us consider a LTI system:
x˙=Ax+Bu\dot{\mathbf{x}} = \mathbf{A}\mathbf{x} + \mathbf{B}\mathbf{u}x˙=Ax+Bu

with control u\mathbf{u}u being the full state linear feedback:
u=ϕ(x)=−Kx\mathbf{u} = \boldsymbol{\phi}(\mathbf{x}) = -\mathbf{K}\mathbf{x}u=ϕ(x)=−Kx

Substitution to system dynamics yields following closed loop relations:
x˙=(A−BK)x=Acx\dot{\mathbf{x}} = (\mathbf{A}-\mathbf{B}\mathbf{K})\mathbf{x} = \mathbf{A}_c\mathbf{x}x˙=(A−BK)x=Ac​x

Thus the control design dedicated to stabilize original system seen as modifying the eigen values of matrix Ac\mathbf{A}_cAc​ such that the resulting system will have required properties (namely stability, response time etc).

This method is called pole-placement (poles - are other names for eigen values).

from scipy.signal import place_poles

Let us test the pole placement technique to stabilize several linear systems.

Controlability

Controllability is an important property of a control system, and the controllability property plays a crucial role in many control problems, such as stabilization of unstable systems by feedback, or optimal control. Roughly, the concept of controllability denotes the ability to move a system around in its entire configuration space using only certain admissible manipulations.

The most widely used notions of controlability was introduce by Kalman with help of following n×nmn\times nmn×nm matrix:

C=[BABA2B...An−1B]\boldsymbol{\mathcal{C}}= \begin{bmatrix} \mathbf{B}& \mathbf{AB}& \mathbf{A^{{2}}B}& ...& \mathbf{A^{{n-1}}B} \end{bmatrix}C=[B​AB​A2B​...​An−1B​]

The theory says that system is full state controllable iff the controllability matrix has full row rank:

rank{C}=n\text{rank}\{\boldsymbol{\mathcal{C}}\} =nrank{C}=n

There are tight relationships between controlability and pole placement technique, if system is controlable one may place poles at any desired locations. In fact in most pole-placement algorithms (for instance celebrated Ackerman formula) one need to invert controllability matrix!

The code that implement above is straight forward:

def ctrb(A, B):
    C = B
    n = np.shape(A)[0]
    # print(n)
    for i in range(1, n):
        A_pwr_n = np.linalg.matrix_power(A, i)
        C = np.hstack((C, A_pwr_n.dot(B)))
    rank_C = np.linalg.matrix_rank(C)

    if rank_C == n:
        test = 'controllable'
    else:
        test = 'uncontrollable'
    return C, rank_C, test

Popov-Belevitch-Hautus Controllability

The alternative view of controlability was provided separately by Hautus, Popov and Belevitch in the 1970-th. it is known as Hautus lemma, also commonly known as the Popov-Belevitch-Hautus test or PBH test.

The lemma is stated as follows, pair {A,B}\{\mathbf{A}, \mathbf{B}\}{A,B} is said to be controllable iff:

rank{[A−ζIB]}=n,∀ζ∈C\text{rank}\{ \begin{bmatrix} \mathbf{A} - \zeta \mathbf{I} & \mathbf{B} \end{bmatrix} \} =n, \quad \forall \zeta \in \mathbb{C}rank{[A−ζI​B​]}=n,∀ζ∈C

However the only way for A−λI\mathbf{A} - \lambda\mathbf{I}A−λI to lose the rank is for λ\lambdaλ to be the eigenvalue of A\mathbf{A}A, thus instead of checking the entire complex plane we may consider just the eigenvalues:

rank{[A−λiIB]}=n,∀i∈{1,2,…,n}\text{rank}\{ \begin{bmatrix} \mathbf{A} - \lambda_i \mathbf{I} & \mathbf{B} \end{bmatrix} \} =n, \quad \forall i \in \{1, 2,\dots, n\}rank{[A−λi​I​B​]}=n,∀i∈{1,2,…,n}

def pbh(A, B):
    lambdas, v = np.linalg.eig(A)
    n = np.shape(A)[0]
    ranks = n*[0]
    test = 'controllable'
    for i in range(n):
      A_e = A - lambdas[i]*np.eye(n)
      M = np.hstack((A_e, B))
      ranks[i] = np.linalg.matrix_rank(M)
      if ranks[i] != n:
        test = 'uncontrollable'
    return lambdas, ranks , test

Stabilizability

A slightly weaker notion than controllability is that of stabilizability. A system is said to be stabilizable when all uncontrollable state variables can be made to have stable dynamics. Thus, even though some of the state variables cannot be controlled (as determined by the controllability test above) all the state variables will still remain bounded during the system's behavior.

Stabilizable systems should be controllable only for the unstable eigenvalues i.e:

rank{[A−λiIB]}=n,∀λ∈{c:Re(c)≥0}\text{rank}\{ \begin{bmatrix} \mathbf{A} - \lambda_i \mathbf{I} & \mathbf{B} \end{bmatrix} \} =n, \quad \forall \lambda \in \{c: \text{Re}(c)\geq0\}rank{[A−λi​I​B​]}=n,∀λ∈{c:Re(c)≥0}

One can slightly modify the controllability test to have following:

def pbh(A, B):
    lambdas, v = np.linalg.eig(A)
    n = np.shape(A)[0]
    ranks = n*[0]
    # M = n*[0]
    unctrl_eigs = []
    test = 'controllable'
    for i in range(n):
        M = np.hstack((A - lambdas[i]*np.eye(n), B))
        ranks[i] = np.linalg.matrix_rank(M)
        if ranks[i] != n:
            unctrl_eigs.append(lambdas[i])

    if np.all(np.array(unctrl_eigs) < 0):
        test = 'uncontrollable but stabilizable'

    else:
        test = 'uncontrollable'

    return lambdas, ranks, test

Thus, for linear systems we can address a variety of issues using numerical techniques based on linear algebra, moreover, the controller synthesis is also performed numerically - with pole placement.

However, the pole assignment is an great technique, but it has two fundamental disadvantages,

  • As the order of the system increases, pole placement becomes a tricky and non-trivial problem since there is no simple mapping between response and the pole locations (partially solved by optimal control routines like LQR which we will discuss in the upcoming classes)
  • This method is applicable to linear systems while most of the real systems are non-linear!

The second problem is actually a fundamental one; The main the question is, can we apply linear control to nonlinear systems? And if yes what kind of tools do we need to analyze the resulting closed loop system.

Nonlinear Systems, Equilibriums

One of the most natural questions ask while studying dynamical systems: Is there any special points or regions such that if system start there will remain there forever? Such regions are called invariant, and in most simple case their represented by so called singular or equilibrium points.

It is particular easy to find equilibriums by looking for such xe\mathbf{x}_exe​ that makes x˙\dot{\mathbf{x}}x˙ vanish, basicly one should just solve for roots of:
f(xe)=0\mathbf{f}(\mathbf{x}_e) = \mathbf{0}f(xe​)=0

Note that for linear systems one have the following:
Axe=0\mathbf{A}\mathbf{x}_e = \mathbf{0}Axe​=0
thus equilibriums are either trivial solution xe=0\mathbf{x}_e = \mathbf{0}xe​=0 or nullspace Null{A}\text{Null}\{\mathbf{A}\}Null{A}

One may always transform the original dynamics to the new variables x~=x−xe\tilde{\mathbf{x}} = \mathbf{x} - \mathbf{x}_ex~=x−xe​ such that:
x~˙=f(x~+xe)\dot{\tilde{\mathbf{x}}} = \mathbf{}\mathbf{f}(\tilde{\mathbf{x}}+\mathbf{x}_e)x~˙=f(x~+xe​)
we call the expression above error or perturbation dynamics

Local Behavior and Linearization

As one note for LTI systems stability is completely determined by the eigenvalues of A\mathbf{A}A,
however in the nonlinear case there are no notion of eigenvalues, a one of the most straightforward ideas
is to approximate the nonlinear system in the vicente of equilibrium point using the first term of Taylor series:

x˙=f(xe)+∂f∂x∣xe(x−xe)\dot{\mathbf{x}} = \mathbf{f}(\mathbf{x}_e)+ \frac{\partial\mathbf{f}}{\partial\mathbf{x}}\Bigr\rvert_{\mathbf{x}_e} (\mathbf{x} - \mathbf{x}_e)x˙=f(xe​)+∂x∂f​​xe​​(x−xe​)

Introducing the error x~\tilde{\mathbf{x}}x~ we may rewrite the equation above in linear form as follows:

\begin{equation*}
\dot{\tilde{\mathbf{x}}} = \mathbf{A}\tilde{\mathbf{x}}
\end{equation*}
where: A=∂f∂x∣xe\mathbf{A} = \frac{\partial\mathbf{f}}{\partial\mathbf{x}}\Bigr\rvert_{\mathbf{x}_e}A=∂x∂f​​xe​​

Thus linearization consist of three steps:

  1. Solving f(xe)=0\mathbf{f}(\mathbf{x}_e) = \mathbf{0}f(xe​)=0 for equilibriums.
  2. Calculation of the system Jacobian with respect to state x\mathbf{x}x
  3. Evaluation of the resulting Jacobian in the equilibrium points xe\mathbf{x}_exe​

The procedure above is known as Lyapunov linearization method and can be used to find the local stability properties of nonlinear systems
nearby equlibrium by studying their linear aproximations as follows:

  • If the linearized system is strictly stable (i.e, Re⁡λi=αi=0,∀i\operatorname{Re}{\lambda_i} = \alpha_i = 0, \forall iReλi​=αi​=0,∀i), then the equilibrium point is asymptotically stable (for the actual nonlinear system).
  • If the linearized system is unstable (∃i,Re⁡λi=αi>0\exists i, \operatorname{Re}{\lambda_i} = \alpha_i > 0∃i,Reλi​=αi​>0), then the equilibrium point is unstable
    (for the nonlinear system).
  • If the linearized system is marginally stable (Re⁡λi=αi≤0,∀i\operatorname{Re}{\lambda_i} = \alpha_i \leq 0, \forall iReλi​=αi​≤0,∀i), then
    one can't conclude anything from the linear approximation (the equilibrium
    point may be stable, asymptotically stable, or unstable for the nonlinear system).

Linearization Based Control over Nonlinear Systems

Similarly if one consider controlled system:
x˙=f(x,u)\dot{\mathbf{x}} = \mathbf{f}(\mathbf{x}, \mathbf{u})x˙=f(x,u)

One may perform the linearization around the operating point:

x~˙=∂f∂x∣xd,udx~+∂f∂u∣xd,ud=Ax~+Bu~\dot{\tilde{\mathbf{x}}} = \frac{\partial\mathbf{f}}{\partial\mathbf{x}}\Bigr\rvert_{\mathbf{x}_d,\mathbf{u}_d} \tilde{\mathbf{x}} + \frac{\partial\mathbf{f}}{\partial\mathbf{u}}\Bigr\rvert_{\mathbf{x}_d,\mathbf{u}_d} = \mathbf{A} \tilde{\mathbf{x}} + \mathbf{B} \tilde{\mathbf{u}}x~˙=∂x∂f​​xd​,ud​​x~+∂u∂f​​xd​,ud​​=Ax~+Bu~

The operating point is found similarly to equilibrium of: f(xd,ud)=0\mathbf{f}(\mathbf{x}_d, \mathbf{u}_d) = 0f(xd​,ud​)=0

Design and analisyse of the linearized system represented by LTI is in fact identical to one you use for linear system.
Thus the controller is still linear state feedback:
u=ud−Kx~\mathbf{u} = \mathbf{u}_d - \mathbf{K}\tilde{\mathbf{x}}u=ud​−Kx~

However you will apply controller not to the linear system which you used to approximate dynamics of actual plant, thus resulting closed loop of original system will still be nonlinear:
x~˙=f(x~,ud−Kx~)=fc(x~)\dot{\tilde{\mathbf{x}}} = \mathbf{f}(\tilde{\mathbf{x}},\mathbf{u}_d - \mathbf{K}\tilde{\mathbf{x}}) = \mathbf{f}_c(\tilde{\mathbf{x}})x~˙=f(x~,ud​−Kx~)=fc​(x~)

Thus the behavior of closed loop system under influence of linear control may be drastically different from the linear one, especially if we move towards the region when linearization fails to provide the accurate description of system behavior (far from linearization point).
